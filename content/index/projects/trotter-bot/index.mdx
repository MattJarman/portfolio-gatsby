A discord bot simple moderation and helpful commands. Using <a className="text-green-600 dark:text-green-400" href="https://store.steampowered.com/">Steam</a>, <a className="text-green-600 dark:text-green-400" href="https://howlongtobeat.com/">HowLongToBeat</a>,
and <a className="text-green-600 dark:text-green-400" href="https://isthereanydeal.com/">IsThereAnyDeal</a>, Trotter can provide
useful commands to check your playtime, find the average completion time, or find the best deal for most games.
